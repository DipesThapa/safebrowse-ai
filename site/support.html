How to contact us

Need Channel Expectations
General questions, onboarding help GitHub Discussions Community response, maintainer follow-up within 5 business days
Feature suggestion Feature Request issue template Reviewed during roadmap planning (quarterly)
Bug report Bug Report issue template with reproduction steps Triage within 3 business days; patch scheduled by severity
Production incident Bug report marked priority + email operations@safebrowse-ai.test Acknowledgement within 1 business day, status updates until resolved
Security vulnerability GitHub Security Advisory or email security@safebrowse-ai.test Initial response within 48 hours, coordinated disclosure process

Operational commitments (best-effort)

  • Maintenance releases at least once per quarter.
  • Hotfixes for critical blocking issues within 3 business days of confirmation.
  • Documentation updates alongside significant UI or behaviour changes.
